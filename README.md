Pelicula Ian
Aplicación web desarrollada con React que consume la API de The Movie Database (TMDb) para mostrar películas populares y sus detalles.

Características
-Lista peliculas populares de cartelera
-Te da el titulo y descripcion de las peliculas
-Muestra imagenes de las mismas que se adaptan a la pantalla
-Consumen una Api externa
-Navegación entre páginas con React Router

Tecnologías utilizadas
-React
-React Router DOM
-Fetch API
-CSS

Deploy
-El proyecto está desplegado en Netlify.
